#   安装依赖    

    cnpm install
    
##  运行程序

    npm start
    
    监听localhost:3000端口
    
##  数据页面

    ./routes/index.js文件里面的对象
    
    如果需要修改页面显示数据，直接添加对象即可
    
    比如:添加一个文煜的图书馆
    
##  文件公共路径

    ejs文件中的./代表public文件夹，
	
	相关内容都放置在该文件夹内
   
## 数据库

    使用的mongodb运行目录下的book.sql,
	
	展示的页面数据均是从数据库中调取的
